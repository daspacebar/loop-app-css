How to use the cancellation flows effectively in just 5 steps
  1. Run exit surveys with help of Loop. 
  2. Do a cancellation reason analysis.

  3. Create a message for subscribers that resonates with the cancellation reason. 
    Can be video (TikTok, IG, YouTube shorts), Blog Embed, Newsletter Embed, Founder's message
  4. Use this customizable HTML space to build your retention campaign. Subscribers can interact with this space when they try to cancel.
  5. Track cancellation attempts & churn rate with Loop analytics
